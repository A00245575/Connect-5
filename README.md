Connect-5

Java Server Client Application.

Requires two clients to Run Game.

///////////// Compile Server ////////////
cd to src folder  --- run javac Connect5Server.java

///////////// Run Server ////////////
cd to src folder  --- run java Connect5Server

///////////// Compile Client ////////////
cd to src folder  --- run javac Connect5Client.java

///////////// Run Client ////////////
cd to src folder  --- run java Connect5Server
